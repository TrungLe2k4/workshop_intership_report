### Week 1 Objectives:
* Connect and get acquainted with the culture and members of the First Cloud AI Journey (FCAJ) program.
* Master core Cloud Computing concepts (IaaS, PaaS, SaaS) and the 5 essential NIST characteristics.
* Understand the AWS Global Infrastructure (Regions, AZs, Edge Locations) and High Availability architectures.
* Grasp the AWS Shared Responsibility Model from a Cybersecurity perspective.

### Achievements:
* Clearly understood the overall architecture of AWS Global Infrastructure and the physical isolation mechanisms between AZs to ensure Fault Tolerance.
* Recognized the security boundaries defined by the AWS Shared Responsibility Model, establishing a strong cybersecurity mindset for cloud deployments.
* Successfully created an AWS Free Tier account to use throughout the internship project.
* Became proficient in navigating the AWS Management Console and switching between global Regions.
